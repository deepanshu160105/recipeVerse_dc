# RecipeVerse – Recipe Search Web App

RecipeVerse is a simple and fast recipe web application that allows users to search recipes, view details, bookmark favourites, and add custom recipes.

---

## About This Project

This project is made to practice JavaScript, API usage, and frontend development skills. It focuses on clean UI and simple architecture.

---

  - views/
  - config.js
  - controller.js
  - helpers.js
  - model.js
- sass/

index.html  
package.json  
README.md  

---

## Features

- Search recipes
- View recipe details
- Bookmark recipes
- Add your own recipe
- Change servings
- Local storage support
- Responsive design

---

## Tech Stack

- HTML  
- CSS / SASS  
- JavaScript (ES6)  
- REST API

## How To Run

Clone the repository:

git clone https://github.com/deepanshu160105/recipeverse.git

Go to folder:

cd recipeverse

Install packages:

npm install

Run the project:

npm start
